# COTORX 定价方案草稿 / Pricing Plans Draft

> 这是初稿，供你调整价格/额度后再定稿。所有额度数字都是基于同类工具（OpusClip、Submagic 等）定价对标 + 咱们产品真实功能推出的建议值，不是最终定论——尤其是**目前代码里还没有任何用量计数/限额机制**，视频拆解、脚本生成、渲染导出目前对所有账号都是"无限次"的。要真正按这套方案上架，还需要一轮开发：给每个账号加用量计数器、按月重置、到达上限后弹窗提示升级。这个我可以在你确定好数字之后接着做。

## 一句话定位

COTORX 不是单纯的字幕/剪辑工具（那是 OpusClip、Submagic 在做的事），而是"TikTok Shop 带货视频的完整生产线"：拆解爆款 → AI 生成带货脚本 → 分镜画布规划 → 手动剪辑出片 → 趋势选品 → 追踪同行。所以定价可以比纯剪辑类工具高一档，卖的是"帮你更快出爆款、更快带货"，不是"帮你加字幕"。

## 定价参考依据 / Benchmarking notes

同类工具 2026 年定价区间（仅剪辑/字幕类工具，功能范围比 COTORX 窄）：
- OpusClip：Free / $15 Starter / $29 Pro / Custom Business
- Submagic：Free(3条水印) / $20 Starter(30条) / $40 Pro(100条) / $80 Agency(300条)

COTORX 覆盖的功能链更长（拆解+脚本+画布+剪辑+选品+追踪一体化），所以定价可以比纯字幕/剪辑类工具略高一档，这份草稿就是按这个逻辑定的。最终数字建议你再结合：(1) 实际获客成本、(2) Claude API + Whisper + ffmpeg 渲染的真实单件成本、(3) 目标客群付费意愿，做一次核算再拍板。

**已按你确认的价格更新：** Starter $29 / Pro $69 / Business $129（月付）。这个价位已经明显高于 OpusClip（$15-29）、Submagic（$20-40）这类单一剪辑工具，定位更接近"全流程内容生产平台"而不是"剪辑插件"，卖点要在落地页上讲清楚"一体化"和"带货导向"这两点，不然用户会拿它跟便宜的剪辑工具比价。

## 上线前还差什么（开发清单）

目前代码里 `accessTier`（business/vip/admin）只控制导航栏可见的 Tab，**不控制次数**，也没有任何按月计数/重置的逻辑。要让上面这套方案真正生效，需要：

1. 数据库加「本月已用次数」字段（按 视频拆解 / 脚本生成 / 渲染导出 / 创作者追踪数量 分别计数），每月自动清零
2. 每个动作触发前检查是否超额，超额则弹窗引导升级
3. 接入真实收款（Stripe 或其他），把付费状态和 tier 绑在一起
4. 定价页 UI（把这份文案套进页面）
